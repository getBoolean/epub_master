Reading order of the EPUB.

This filters out auxiliary items.

Arguments:

  • includeAuxiliary Auxiliary items may be optionally treated as primary items. If left as false, (the default), the auxiliary content documents should be rendered upon activation (such as through a hypertext link or entry in NCX), in some manner distinct from the main reading order

Implementation

List<EpubSpineItemRef> getReadingOrder({
  bool includeAuxiliary = false,
}) {
  if (includeAuxiliary) {
    return itemRefs;
  }

  return itemRefs.where((itemRef) => itemRef.isPrimary).toList();
}
